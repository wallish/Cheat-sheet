# SQL cheat

### Add column
```sql
ALTER TABLE Users ADD foo INT(2) DEFAULT NULL AFTER bar;
ALTER TABLE Users ADD foo TINYINT(1) DEFAULT NULL AFTER bar;
ALTER TABLE Users ADD foo VARCHAR(255) DEFAULT NULL AFTER bar;
ALTER TABLE Users ADD foo TEXT DEFAULT NULL AFTER bar;
ALTER TABLE Users ADD foo DATE DEFAULT NULL AFTER bar;
```

### Empty
```sql
TRUNCATE TABLE Users;
ALTER TABLE Users AUTO_INCREMENT = 1;
```
